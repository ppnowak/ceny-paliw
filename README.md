# ⛽ Ceny Paliw

Maksymalne ceny detaliczne paliw w Polsce (PB95, PB98, ON), ogłaszane przez Ministra Energii.

Dane pobierane automatycznie ze strony [gov.pl/web/energia](https://www.gov.pl/web/energia/wiadomosci) i publikowane jako statyczna strona na GitHub Pages.

## Uruchomienie lokalne

```bash
npm install
npx http-server site -p 8080
```

## Pobranie danych

```bash
npm run scrape
```
